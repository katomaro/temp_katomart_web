Hotmart Cloudfront Embed Downloader

por Katomaro

Sobre a Ferramenta

O objetivo desta ferramenta é realizar o download de vídeos hospedados pelo player da Hotmart que são entregues via AWS CloudFront em sites externos (fora da área de membros principal). Muitas extensões de navegador bloqueiam o download de vídeos, por isso esta ferramenta foi criada é um atalho manual com pretensão de ser funcional para quem se encontra em cenário de desespero.

Como Usar

1. Instale as dependências Python

Navegue até a pasta do projeto e execute o comando abaixo no seu terminal:

pip install -r requirements.txt

2. Obtenha o URL do Manifesto (.m3u8)

Abra o site com o vídeo, ative as ferramentas de desenvolvedor (F12), vá para a aba "Rede" (Network), filtre por master, clique com o botão direito da requisição do manifesto principal, e com copiar > URL como mostrado abaixo:
